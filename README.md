# PDF Pipeline v3

Pipeline PDF robuste et observable.

## Installation

```bash
pip install -r requirements.txt
```

## Exécution

Ajoute un fichier nommé:

```text
doc.pdf
```

Puis :

```bash
python main.py
```

## Notebook

Le notebook:

```text
pdf_pipeline_demo.ipynb
```

permet :
- tests
- debugging
- analyse qualité
- inspection DataFrame
